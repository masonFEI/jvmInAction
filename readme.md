## java代码执行流程
1. 编译（前端编译，与执行引擎中的编译成热点代码（后端编译）形成区分），形成字节码文件
2. 字节码文件运行在jvm之上

## jvm的架构模型
> 一种是基于栈的指令集架构（Hotspot），另一种是基于寄存器的指令集架构。
> 基于栈的指令集架构以零地址为主，基于寄存器的指令集架构都以一地址指令，二地址指令，三地址指令为主
>
> .class文件反编译的命令为 javap -v xxx.class
> 
>  jps 查看执行中的进程
